# WARP-PLUS-HKG
  将Cloudflare WARP的主机托管中心锁定为HKG(香港), 以换取低延时warp+.

## 下载
'https://github.com/Windla/WARP-PLUS-HKG/archive/refs/heads/main.zip'

## 请确保有这些库
pip install requests

pip install pywin32

pip install pyuac

pip install ping3

## 如何使用
  需要Python 下载'https://python.org/'
  安装Python后双击运行即可

## 关于UAC
  UAC是为了开关Cloudflare WARP服务使用, 无其他用处！
